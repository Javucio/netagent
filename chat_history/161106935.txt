Bot: Un router y un switch son dos dispositivos diferentes en una red, con funciones y propósitos únicos. Un router es un dispositivo que se encarga de enrutar el tráfico de red, es decir, de dirigir el tráfico de datos entre redes diferentes o dentro de una red. Un switch, por otro lado, es un dispositivo que se encarga de conectarse y desconectarse de dispositivos en una red local, permitiendo la comunicación entre ellos.

Bot: El modelo OSI (Open Systems Interconnection) es un modelo de comunicación de red que se divide en 7 capas. Estas capas son:

1. Física (Physical): se ocupa de la transmisión de datos a través de cables o fibra óptica.
2. Enlace de datos (Data Link): se encarga de la formación y recepción de paquetes de datos.
3. Red (Network): se ocupa de la ruta que los paquetes de datos deben seguir para llegar a su destino.
4. Transporte (Transport): se encarga de asegurar que los paquetes de datos lleguen intactos al destinatario.
5. Sesión (Session): se ocupa de establecer, mantener y finalizar la conexión entre dos dispositivos.
6. Presentación (Presentation): se encarga de codificar y decodificar los datos para que sean legibles por los dispositivos.
7. Aplicación (Application): se ocupa de proporcionar servicios de aplicación, como el envío y recepción de archivos, correo electrónico, etc.

Cada capa interactúa con la capa superior y la capa inferior para realizar su función. Esto permite una comunicación eficiente y segura entre dispositivos de red.
